# EstateIQ Product Metrics

## Activation Metrics

- Pilot organization created.
- First property added.
- First user signs in by role.
- First inspection started.
- First inspection completed.
- First homeowner report viewed.
- First PDF downloaded.
- First maintenance issue created.
- First owner update shared.

## Engagement Metrics

- Login frequency by role.
- Inspections completed per property.
- Reports viewed per homeowner.
- PDFs downloaded per inspection.
- Photos uploaded per inspection or maintenance issue.
- Most-used workflows.
- Least-used workflows.
- Mobile versus desktop usage.
- Time spent inactive on key screens.

## Retention Metrics

- Weekly active pilot organizations.
- Repeat inspections after first use.
- Repeat report views by homeowner.
- Maintenance follow-up completion.
- Scheduled visit completion.
- Feedback submitted after initial onboarding.

## Operational KPIs

- Average inspection completion time.
- Average photos per inspection.
- Percentage of reports with executive summaries.
- Percentage of inspections with urgent issues.
- Open maintenance item count.
- Maintenance resolution time.
- Vendor assignment rate.
- Owner update sharing rate.

## Pilot Success Signals

- Customer uses EstateIQ without a guided walkthrough.
- Inspector completes a real field workflow on mobile.
- Homeowner report is sent externally.
- Customer requests additional properties or users.
- Customer asks about pricing.
- Customer feedback shifts from basic usability to workflow expansion.

## Metrics To Avoid Overvaluing Early

- Total page views.
- Vanity demo clicks.
- Large feature wish lists.
- AI usage without human approval.
- Enterprise integrations before workflow-market fit.
