# EstateIQ Documentation

EstateIQ documentation is organized by audience and purpose.

## Folders

- `founder/`: Plain-English operating guides for the CEO/founder.
- `product/`: Product strategy, metrics, roadmap, design system, analytics, and AI operating rules.
- `architecture/`: Technical architecture, deployment, and project structure.
- `sales/`: Future sales scripts, pricing notes, objection handling, and customer-facing collateral.
- `marketing/`: Future website copy, positioning, launch plans, and brand messaging.
- `pilots/`: Pilot program, pilot readiness, and manual testing documentation.
